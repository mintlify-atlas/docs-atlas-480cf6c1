# Welcome to Verbalize

Verbalize is a powerful web-based assistant designed specifically for technical writers who want to ensure their documentation adheres to industry-standard style guides. Built with [Vale](https://vale.sh/), the leading prose linting engine, Verbalize provides instant feedback on your writing to help you maintain consistency, clarity, and professionalism.

## What is Verbalize?

Verbalize combines a modern, intuitive interface with the powerful Vale linting engine to create a seamless writing and validation workflow. Write your documentation in Markdown, preview it in real-time, and receive immediate feedback on style violations—all in a single, unified workspace.

## Who is Verbalize For?

Verbalize is designed for anyone who writes technical documentation:

- **Technical Writers** crafting API documentation, user guides, or tutorials
- **Developer Advocates** creating blog posts and developer resources
- **Documentation Teams** maintaining consistency across large documentation sets
- **Open Source Contributors** ensuring contributions meet project style standards
- **Content Teams** enforcing brand voice and style guidelines

## Key Features at a Glance

<Info>
  Verbalize processes your documentation entirely server-side, with support for documents up to 1MB in size. The Vale engine runs on-demand, ensuring your text is validated against the latest style guide rules.
</Info>

### Three-Column Workflow

Verbalize's interface is organized into three synchronized panels:

1. **Editor** - Write your Markdown documentation with character count tracking
2. **Preview** - See rendered output with synchronized scrolling
3. **Linting Alerts** - View detailed style violations with severity levels (error, warning, suggestion)

### Detailed Diagnostics

Every linting alert includes:

- **Line number** and precise text span location
- **Severity level** (error, warning, or suggestion)
- **Clear message** explaining the style issue
- **Rule name** for reference and learning
- **External links** to style guide documentation for context

### Dark Mode Support

Built with `next-themes`, Verbalize automatically adapts to your system preferences and includes a manual theme toggle for comfortable writing in any lighting condition.

## Architecture Overview

Verbalize is built on modern web technologies:

<CodeGroup>
```typescript Frontend
// Next.js 16 with App Router
// React 19 for UI components
// Tailwind CSS for styling
// react-markdown for preview rendering
// Lucide React for icons
```

```typescript Backend
// Next.js API Routes for serverless functions
// Vale CLI for prose linting
// Temporary file handling for Vale integration
// Support for custom rule injection
```
</CodeGroup>

The application uses a **request-response architecture** where:

1. User text is sent to `/api/lint` endpoint
2. A temporary file is created for Vale to process
3. Vale binary executes with dynamic configuration
4. Results are parsed and returned as JSON
5. Frontend displays diagnostics with clickable navigation

<Note>
  The Vale binary is automatically installed during `npm install` via a post-install script (`scripts/install-vale.js`), ensuring the linting engine is always available.
</Note>
